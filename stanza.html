Stanza

WebApp, MobileApp, and BackEnd

Overview

Stanza is a music discovery app that shows interesting and unique bite-size lyrics to users. Once they've found one that they like, users can share with the lyric with their friends, get linked to listen to the song on their favorite music streaming service, or add it to their favorites.

Initially the project began with a front end engineer who I directed. Due to personal reasons the front end engineer left and the front end was completed by me.

Web App and Back End

The web app was designed for admin to easily sift through songs and identify lyrics to categorize according to genre, category, and decade. The UI was designed to be fluid and intuitive. Functionality was provided to allow editing, viewing, and deleting to pre-existing lyric items. Lastly we added a Dashboard that presented the metrics that we tracked.

The back end was made to serve both users (mobile app) and admin (web app). The mobile app used back end endpoints to allow users to favorite and flag songs. The web app used back end endpoints to allow admin to add, edit, delete, and view lyric items. Some other features of the back end include making third party API requests, keeping the API keys secure, tracking analytics, and more.

Notable Features

  • Designed the architecture between the web app, back end, and mobile app
  • User authentication for multiple admin
  • A dashboard to display database analytics
  • Connection with spotify and genius api to serve admin with content to add to database
  • Ability to add, delete, edit, and read lyric items
  • GraphQL back end to handle all data requests
  • Developed a way to economically(free) push updates

Phone App

The Stanza mobile app serves bite size lyrics to users tinder style. Each lyric can be liked, shared, and connected to your favorite streaming service. The app also has multiple UI themes and text sizes to choose from.

Notable Features

  • MongoDB(Realm) local database for content
  • Favorite and flag lyrics through back end
  • Use of native share to share the songs you love
  • Link to listen on your favorite music streaming app
  • A selection of themes and text sizes
  • Explicit lyric filtering, and notifications
